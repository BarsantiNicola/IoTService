The package contains all the classes used to define and send statistics to the web pages.
It contains two main classes:

    - Statistic: class to define the single element to be showed it is composed by two values which describe the
                 two values needed to be shown on a cartesian graph(Date,value)

    - Statistics: container for Statistic instances, it maintains all the values that will compose the graph
